# Open Circuit Benchmark (OCB) Dataset

This directory contains the **Open Circuit Benchmark (OCB)** dataset, used as the experimental foundation for CktGen.

## üîó Appreciation & Source

The OCB dataset was introduced by the authors of **CktGNN** ([Dong et al., 2023](https://github.com/zehao-dong/CktGNN)). Their work established a crucial standard for cataloging and benchmarking analog circuits. We are honored to build upon this high-quality foundation.

- **Foundational Paper**: [CktGNN: Circuit Graph Neural Network for Electronic Design Automation](https://arxiv.org/abs/2308.16406)
- **Official Repository**: [https://github.com/zehao-dong/CktGNN](https://github.com/zehao-dong/CktGNN)
